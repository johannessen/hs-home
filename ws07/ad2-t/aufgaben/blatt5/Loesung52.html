Aufgabe 5-2

(Übungsblatt 5)

Bestimmen Sie den Wahrheitsgehalt folgender Java-Ausdrücke vom Typ boolean (mit den Variablen a und b deklariert mit Typ int).

  1. a + b == b + a
  2. true || false && true
  3. ! true || a != a
  4. a == a + 1
  5. false || 10 <= (8 * 2) % 10 + 5
  6. Zusatzaufgabe: 1 > 0x100 >> 8

Lösungsvorschlag

zu den Teilaufgaben:

  1. true (Kommutativität)
  2. true (= true || false)
  3. false (= false || false)
  4. false (Identität)
  5. true (= false || true = false || 10 <= 11)
  6. false (= 1 > 1 = 1 > 0x1)

Anmerkung. Bei Problemen dieser Art ist das „Code Pad“ in BlueJ äußerst hilfreich. Siehe Unterlagen.

Arne Johannessen, 28. November 2007
